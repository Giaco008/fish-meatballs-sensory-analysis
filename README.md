# Tamarong Fish Meatballs Sensory Analysis

## 📌 Project Overview
This project analyzes the sensory quality and acceptability of Tamarong (Selar crumenophthalmus) fish meatballs infused with Spinach (Spinacia oleracea) and Moringa (Moringa oleifera).

## 🎯 Objectives
1. Evaluate sensory attributes:
   - Color
   - Aroma
   - Texture
   - Taste
   - Aftertaste
   - Overall Acceptability

2. Determine if there is a significant difference between formulations.

## 🛠 Tools Used
- Python (Pandas, NumPy, SciPy)
- Matplotlib / Seaborn
- Statistical Analysis (ANOVA, Post-hoc tests)

## 📊 Analysis Performed
- Descriptive statistics (mean, standard deviation)
- Heatmap visualization of sensory attributes
- Radar chart for product comparison
- Bar graphs for average scores
- Boxplots for distribution analysis
- One-way ANOVA
- Post-hoc multiple comparisons
